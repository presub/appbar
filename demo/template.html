An <x-appbar> is a simple component using flexbox to emulate the standard layout of a header seen at the top of a mobile application.

Basic usage

You can use any elements in place of the <button> tags in the example below. A heading <h1> to <h6> element is required and will be created automatically if not provided.

Markup

<x-appbar>
    <button>=</button>
    <h1>Title</h1>
    <button>+</button>
    <button>?</button>
</x-appbar>
